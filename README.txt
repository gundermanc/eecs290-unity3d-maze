A collaborative project for Case Western Reserve University's EECS 290
class.


Project Features:
* Develop maze game in Unity 3D
* Add first person shooting and swords
* Start game screen
* Menus
* Health System
* Flashlight System with batteries
* Health Pack Powerups
* Levels
* Throwing Knives
* Enemies
* Onscreen Display
* Music
* Sound Effects
* Attack feedback
* etc.
